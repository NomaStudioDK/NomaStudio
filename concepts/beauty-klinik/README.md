# Nord Studio Beauty Koncept

Dette er en separat konceptmappe til et fiktivt beauty-/klinikprojekt:
`Nord Studio Beauty`.

Konceptet kan senere vises som portfolio-/case-materiale på Noma Studio-sitet,
men bør tydeligt markeres som et konceptprojekt og ikke en lanceret kundeside.

Filer:

- `index.html` - konceptets forside med intro, ydelser, booking og åbningstider
- `ydelser.html` - separat side med mere dybdegående beskrivelser
- `style.css` - styling kun til beauty-klinik-konceptet
- `script.js` - booking-demo og mobilmenu
- `assets/nord-studio-beauty-logo.png` - logoet fra brugerens reference
